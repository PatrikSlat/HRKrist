# HRKrist

HRKrist je web aplikacija posvećena promicanju kršćanske vjere i kulture u Hrvatskoj kroz sadržaje poput molitvi, biblijskih čitanja, crkvenog kalendara i duhovnog savjetovanja.

## Tehnologije korištene u projektu

- **Backend**: Node.js (Express.js)
- **Baza podataka**: MySQL
- **Frontend**: Vue.js s Quasar Frameworkom

## Značajke

- **Molitve**: Kolekcija različitih kršćanskih molitvi za razne prigode.
- **Biblijska čitanja**: Dnevna čitanja iz Biblije.
- **Organiziranje humanitarnih evenata**
